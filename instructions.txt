Чтобы создать и подвязать bash-скрипт к C++ коду для обработки текста, выполните следующие шаги:

### 1. Напишите C++ программу (`text_processor.cpp`)
Пример программы, которая переводит текст в верхний регистр:
```cpp
#include <iostream>
#include <string>
#include <cctype>

int main() {
    std::string input;
    while (std::getline(std::cin, input)) {
        for (char &c : input) {
            c = std::toupper(static_cast<unsigned char>(c));
        }
        std::cout << input << std::endl;
    }
    return 0;
}
```

### 2. Скомпилируйте C++ программу
```bash
g++ text_processor.cpp -o text_processor
```

### 3. Создайте bash-скрипт (`process_text.sh`)
```bash
#!/bin/bash

# Проверка наличия исполняемого файла
if [ ! -f "./text_processor" ]; then
    echo "Ошибка: файл 'text_processor' не найден!"
    exit 1
fi

# Обработка ввода
if [ $# -eq 0 ]; then
    # Режим stdin
    ./text_processor
else
    # Режим аргументов (объединяем все аргументы в одну строку)
    echo "$*" | ./text_processor
fi
```

### 4. Дайте права на выполнение скрипту
```bash
chmod +x process_text.sh
```

### Как использовать:
#### Запуск с аргументами
```bash
./process_text.sh "привет мир"
```
**Вывод:**
```
ПРИВЕТ МИР
```

#### Запуск с вводом из stdin
```bash
echo "hello world" | ./process_text.sh
```
**Вывод:**
```
HELLO WORLD
```

#### Интерактивный режим
```bash
./process_text.sh
```
(введите текст с клавиатуры, завершите ввод с помощью `Ctrl+D`)

### Как это работает:
1. **C++ программа**:
   - Читает текст из стандартного ввода (построчно)
   - Преобразует текст в верхний регистр
   - Выводит результат

2. **Bash-скрипт**:
   - Автоматически определяет источник ввода (аргументы или stdin)
   - Передает текст в C++ программу через пайп
   - Обрабатывает ошибки (например, отсутствие бинарного файла)

### Кастомизация:
1. **Измените полезную нагрузку** в C++ коде (например, добавив шифрование, анализ текста и т.д.):
```cpp
// Пример: инвертирование регистра
for (char &c : input) {
    if (std::islower(c)) c = std::toupper(c);
    else if (std::isupper(c)) c = std::tolower(c);
}
```

2. **Расширьте скрипт** для поддержки файлов:
```bash
# Добавьте в секцию обработки аргументов:
if [ "$1" = "-f" ]; then
    cat "$2" | ./text_processor
fi
```

### Важные примечания:
- Убедитесь, что C++ программа и bash-скрипт находятся в одной директории
- Для сложной обработки текста используйте буферизацию вывода в C++
- Добавьте обработку ошибок в C++ для некорректного ввода

Пример работы с файлом через скрипт (после кастомизации):
```bash
./process_text.sh -f input.txt
```
